# Golden Atlas Players Website

Official website project for the **Golden Atlas Players**, a Rocky Horror shadowcast performing at the GAP Theatre in Wind Gap, Pennsylvania.

## Tech Stack

- Astro
- HTML/CSS
- GitHub
- (Future) Cloudflare Pages

## Project Goals

- Community website
- Show information
- Cast information
- Discord integration
- Facebook integration
- Photo gallery
- Mobile-friendly design
